  <br/>
  
  * In order to upload files to **minio** server we need to install another plugin named
  **glue-plugin-storage** which uses minio, to upload files and store them.
  
  <br/>
  
  * In order to start this plugin we need to run the following command:
    
    <br/>
  
    ```bash
      $ node glue develop:up
    ```
  
    <br/>

    This command will start running the docker image of all the instances
    and plugin we have created. Here, in this case it will start running
    our **minio plugin instance**.
  
  <br/>

* ## Instructions to start development with Glue Plugin Minio
  
  <br/>
  
  * In order to interact with our **minio** server to upload and store files,
  we can use another plugin named **glue-plugin-storage** which already comes
  with configuration with our minio server and we can start using that plugin
  to our app.

  <br/>

  * If you want to interact with the minio server by yourself then you can simply
  pick up the **configuration information** provided in the console when we have
  installed the **minio plugin**.
